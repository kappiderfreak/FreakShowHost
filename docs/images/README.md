## Vor einem öffentlichen Release · Before going public

**DE** – Aufnahmen vor einer Veröffentlichung erneut prüfen. Zu entfernen/prüfen: Passwörter,
OAuth-Tokens, Client-IDs; vollständige persönliche Widget-Links; nicht als Beispiel gedachte
IP-Adressen; Twitch-Name, Follower-/Abo-Zahlen, Chatverläufe; lokale Pfade; Medien ohne
Veröffentlichungserlaubnis.

**EN** – Re-check screenshots before publishing. Remove/verify: passwords, OAuth tokens, client
IDs; full personal widget links; IP addresses not meant as examples; Twitch name, follower/sub
counts, chat logs; local paths; media without permission to publish.
